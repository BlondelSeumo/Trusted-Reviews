A) Installing the MySQL Database - top

  • Go to your hosting Control Panel and point to the database manager
  • Create a MySQL database and ASSIGN a MySQL User to it.
    Go to phpMyAdmin and select the database you've just created.
    Import the file located into "database/reviews.sql" folder.
  • Step two is to open .env (located in the root/main/parent folder of the script) with any text editor and update mysql database connection info
  • Note: .env file or .htaccess file ( dot files ) might be hidden in your operation system.
    You can reveal them depending on the OS you're using.
    • LINES 1 TO 5


B) Installing the Script - top

  • Connect to your hosting and upload the extracted zip file folder of the script to your domain. (must be under either root or subdomain. No subfolders please.)
  • Make sure .htaccess file is included otherwise the script will mallfunction as the file is required to do the mod_rewrite SEO friendly URL's job

C) Configure Admin Credentials - top

Just open the same .env file ( seen at step 1 ) and enter your admin user/pass

D) Configure Title, Analytics Code, Site Logo, etc. - top

  • Navigate to your-website-url.com/admin/login and login with the credentials you've just configured earlier at step C).
  • Point to "Configuration" tab
  • You will see a nice & complete form with options to set the following: SEO Title, Site Title, Insert Analytics Code (like Google Analytics JS Snippet, etc), Site Logo and Homepage related configuration options.

D) Configure Payments API - top

PHP Trusted Reviews supports two gateways for taking payments:

  • Stripe
  • PayPal

STRIPE API

  • Step one is to go to Stripe account settings

  • Step two is to go to Admin->Configuration and enter these two credentails.

PayPal API

With PayPal it's now even easier by simply entering your PayPal Email address in Admin->Configuration

E) Configure Mail Server - top

  • Navigate to your-website-url.com/admin and login with the credentials you've just configured earlier at step C).
  • From the left side click "Mail Server"
  • You'll be able to select between default mail() function and SMTP Server (recommended)

H) Configure Plans Pricing & Currency - top

  • Navigate to your-website-url.com/admin and login with the credentials you've just configured earlier at step C).
  • Point to Ads Setup.

F) Manage Ad Codes - top

  • Navigate to your-website-url.com/admin and login with the credentials you've just configured earlier at step C).
  • Point to Settings -> Ads Tab.
  • You can enter any ad codes you prefer (like Google Adwords, etc.)

G) Manage Privacy / ToS Pages - top

  • Navigate to your-website-url.com/admin and login with the credentials you've just configured earlier at step C).
  • Select General Pages from the left side menu.

G) Bulk Import - top

You have the option to upload a CSV file with a list of companies to better launchstart your website instead of going with an empty one.
For the you can go to ADMIN->BULK UPLOAD and upload a file with this format:
ATTENTION: If you don't follow the format specified STRICTLY, it will fail of course. Latitude and Longitude is a must for location radius filtering
url, business name, latitude, longitude, location, category
example: 
crivion.com, "crivion web", 51.5010, 0.1933, "Kensignton, London, UK", "Computers, accessories, and services."

G) Crucial: Google Places API - top

In order for location filtering to work and many other functions like address autocomplete when submitting a company you'll need to enable and add the Google Maps/Places API Keys.
Without this, the app will not function properly.

Obtain Google Places API KEY
Go to https://developers.google.com/places/web-service/get-api-key

Also, you can watch this tutorial on youtube I've found useful on how to generate google places api key: https://www.youtube.com/watch?v=zBur_hgFFRc
After obtaining the key, just go to ADMIN->Configuration and enter it in the GMAPS PLACES API KEY tab

Congrats, if you reached this step and completed everything you'll now enjoy a fully working platform
